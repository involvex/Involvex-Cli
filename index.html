üìñ Usage Examples

Update All Package Managers

1. Launch InvolveX.Cli.exe 2. Navigate to "Update" menu 3. Select "Update All" option 4. Watch progress as all package managers are updated

Clear System Cache

1. Select "Cache" from main menu 2. Choose "Clear Cache" option 3. Confirm the operation 4. System caches are cleared automatically

Network Diagnostics

1. Choose "Network" from main menu 2. Select "Run Ping Test" 3. Enter hostname (e.g., google.com) 4. View ping results and latency information
